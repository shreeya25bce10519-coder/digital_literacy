# Anomaly-based-Intrusion-Detection-System
This System is a technical implementation of the "Zero Trust" mindset required for modern digital citizenship. This project moves  by using an Isolation Forest algorithm to actively verify the integrity of a system rather than trusting its surface-level appearance. The system functions by first normalizing data through feature scaling, ensuring that all digital inputs are weighted equally for mathematical analysis. It then employs an ensemble of decision trees to isolate anomalies—points that are "few and different"—which represent potential security threats like  unauthorized access. By generating a visual scatter plot that highlights these risks in red against the normal blue data, the tool provide the necessary "Cyber Awareness" to fill the gap between theoretical knowledge and practical defense. Ultimately, this project serve as a professional alternative for a student's skills, demonstrating a commitment to systemic responsibility and the ethical grounding needed for a complex digital environment.
